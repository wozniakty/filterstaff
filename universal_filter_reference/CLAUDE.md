# Universal Filter Reference

This folder contains a direct copy of a very popular filter template provided by a LE Community member and streamer.

All credit for this filter template go to [Raxxanterax]https://www.twitch.tv/raxxanterax

The files have been included manually in here to simplify reference when developing.

`Raxx S4 Ultimate Filter.xml` - The raw template used by players to set up their universal filter.
`Raxx S4 Ultimate Filter Bladestorm Rogue & Fire Aura Mage.xml` - An example of a filled out template filter Raxx used to farm for gear for 2 classes in latest season.
